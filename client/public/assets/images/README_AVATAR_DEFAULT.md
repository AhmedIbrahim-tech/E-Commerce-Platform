# Default Avatar Image

## Location
`/assets/images/avatar-default.png`

## Requirements
- Professional, neutral silhouette
- Light background
- Circular-friendly design
- Suitable for e-commerce platform
- Recommended size: 200x200px or larger (square)

## Usage
This image is used as a fallback when:
- User has no profile image (`profileImageUrl === null`)
- Profile image fails to load

## Note
If this file doesn't exist, the application will show a broken image. Please add a professional default avatar image at this location.
